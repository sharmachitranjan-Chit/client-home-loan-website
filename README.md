# Home Loan DSA Starter Website

A zero-build, mobile-friendly static website for a home-loan DSA/facilitation business.

## Fast customisation

Open `assets/js/config.js` and replace:
- business name and tagline
- phone and WhatsApp number
- email
- service area
- website URL
- about and fee wording

Then replace `https://www.example.com/` in:
- `index.html`
- `robots.txt`
- `sitemap.xml`

Also update the contact details and business-specific wording in:
- `privacy.html`
- `terms.html`

## Local preview

Double-tap `index.html` on a computer, or open it in a browser/file-manager app on Android. For the most reliable preview, use GitHub Pages, Cloudflare Pages, or a local static server.

## Cloudflare Pages settings

This project requires no build command.

- Framework preset: None
- Build command: leave blank
- Build output directory: `/`
- Production branch: `main`

## Important launch checklist

- Replace every placeholder.
- Confirm actual lender tie-ups and products.
- Add the DSA's legal/business name and address.
- Review privacy notice, terms, fees and grievance contact.
- Do not claim guaranteed approval, lowest rate or RBI registration unless literally true and verifiable.
- Do not collect Aadhaar/PAN/bank statements through an unsecured public form.
- Test phone, email, WhatsApp, calculator and all navigation links.
